Listas y tablas

Listas

Listas ordenadas

Las listas ordenadas son las listas que muestran los elementos en un orden

<!--Etiqueta de listas ordenadas-->
<!--type="A,a,I,i"-->
<ol>
    <!--Etiqueta para agregar elementos-->
    <li>Elemento uno</li>
        <ol>
            <li>Subelemento uno</li>
            <li>Subelemento dos</li>
        </ol>
    <li>Elemento dos</li>
</ol>
  1. Elemento uno
    1. Subelemento uno
    2. Subelemento dos
  2. Elemento dos

Listas desordenadas

Las listas desordenadas sirven para mostrar o listar elementos sin un orden.

<!--Etiqueta de listas desordenadas-->
<!--type="circle,disc,square"-->
<ul>
    <li>Elemento uno</li>
    <li>Elemento dos</li>
</ul>
  • Elemento uno
  • Elemento dos

Tablas

Una tabla es una herramienta para organizar la información en filas y columnas.

<!--Etiqueta de tablas-->
<!--border="solid"-->
<!--valign="top, middle, bottom"-->
<table>
<!--Representa filas-->
<tr>
    <!--Representa encabezados-->
    <th>Titulo 1</th>
    <th>Titulo 2</th>
</tr>
<tr>
    <!--Representa columnas-->
    <td>Celda 1</td>
    <td>Celda 2</td>
</tr>
</table>
Titulo 1 Titulo 2
Celda 1 Celda 2
Celda 3 Celda 4
