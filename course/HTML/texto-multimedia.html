HTML (HyperText Markup language)

El HTML se utiliza para crear paginas web por medio de etiquetas estructurando los elementos que las componen, las cuales son interpretadas por el navegador permitiendo su visualización segun como se hayan organizado.

Etiquetas

Las etiquetas delimitan un fragmento de contenido para darle un proposito en el sitio web.

Componente Descripción
Elemento Hace parte de la página.
Etiqueta de apertura Consiste en el inicio del elemento.
Etiqueta de cierre Consiste en el fin del elemento.
Contenido Contenido del elemento.

Estructura HTML

<html>
    <head>
        <!--Cabeza del sitio web-->
    </head>
    <body>
        <!--Cuerpo del sitio web-->
    </body>
</html>

Etiquetas de encabezados

Solo debe haber una etiqueta <h1> por cada documento HTML.

<h1>Encabezado principal del sitio web</h1>
<h2>Encabezado de segundo nivel</h2>
<h3>Encabezado de tercer nivel</h3>
<h4>Encabezado de cuarto nivel</h4>
<h5>Encabezado de quinto nivel</h5>
<h6>Encabezado de sexto nivel</h6>

  • Encabezado de segundo nivel

  • Encabezado de tercer nivel

  • Encabezado de cuarto

  • Encabezado de quinto nivel
  • Encabezado de sexto nivel

Etiquetas multimedia

Los elementos multimedia deben ir dentro de una etiqueta <figure>.

<!--Etiqueta contenedor de figuras-->
<figure>
    <!--Etiqueta de imagen-->
    <!--width="Ancho" - height="Alto" (px - #%)-->
    <img src="Fuente" alt="Texto alternativo">
    <figcaption>Descripción figura</figcaption>
    <!--Etiqueta de contenido web-->
    <iframe Contenido web embebido></iframe>
    <!--Etiqueta de audio-->
    <audio></audio>
</figure>
