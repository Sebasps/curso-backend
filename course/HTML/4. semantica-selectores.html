Etiquetas semanticas y selectores HTML

Selectores

Los selectores permiten identificar distintos elementos en el documento HTML para hacer cambios en especifico.

Selector Descripción Identificación Utilización
ID Un mismo ID solo lo debe tener un elemento por documento. <etiqueta id="id_name"> #id_name{}
Class Las clases se pueden utilizar en multiples elementos. <etiqueta class="class_name"> .class_name{}

Para utilizarlos en estilos se requiere una hoja de estilos externa o la etiqueta <style> dentro de <head>.

<!--Selector ID-->
<p id="parrafoUnico">Elemento unico</p>
<!--Selector Class-->
<p class="parrafoMultiple">Elemento multiple 1</p>
<p class="parrafoMultiple">Elemento multiple 2</p>

#parrafoUnico{
    width: fit-content;
    background-color: darkblue;
    font-size: 20px;
    color: tomato;
    text-align: center;
}
.parrafoMultiple{
    width: fit-content;
    background-color: turquoise;
    font-size: 15px;
    color: purple;
    text-align: center;
}

Etiquetas semanticas

<!--El <header> está predefinido para guardar la información principal del sitio (nombre, logo y menú de navegación)-->
<header>
    <!--Menú de navegación-->
    <nav>
        <ul>
            <li><a href="enlace_1">Sección_1</a></li>
            <li><a href="enlace_2#id_name">Sección_2</a></li>
            <li><a href="#id_name">Sección_3</a></li>
        </ul>
    </nav>
</header>
