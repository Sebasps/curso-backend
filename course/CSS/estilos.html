CSS (Cascading Style Sheets)

Es una hoja de estilo en cascada que va aplicando formato de forma descendente a los elementos en la página web.

Selectores

Selector Descripción Identificación Utilización
Etiqueta Hace referencia a cualquier elemento que posea la misma etiqueta. <tag> tag{}
ID Un mismo ID solo lo debe tener un elemento por documento. <tag id="id_name"> #id_name{}
Class Las clases se pueden utilizar en multiples elementos. <tag class="class_name"> .class_name{}

Tipografía

Propiedad Descripción
font-family Cambia el tipo de letra del texto.
font-size Cambia el tamaño de letra del texto.
color Cambia el color del texto.

Background

Propiedad Descripción
background-color Color de fondo.
background-image Imagen de fondo.
background-position Posición del fondo.
background-size Tamaño del fondo.
background-repeat Repetición del fondo.

Estilos de texto

Propiedad Descripción
text-align Alineación del texto.
text-decoration Estilos del texto.
text-transform Transformación del texto.
text-indent Identación.
letter-spacing Espacio entre letras.
line-height Espacio entre líneas.

Modelo de caja

Content

Propiedad Descripción
width Modifica el ancho.
height Modifica el alto.
max-width
min-width		 Define el máximo o el mínimo ancho.
max-height
min-height		 Define el máximo o el mínimo alto.

Border

Propiedad Descripción
border-color Modifica el color del borde.
border-style Modifica el estilo del borde.
border-width Modifica el ancho del borde.
border Agrupa las propiedades de border.

Padding

Propiedad Descripción

padding-right		 Establece relleno a un elemento en derecha o izquierda.
padding-top
padding-bottom		 Establece relleno a un elemento arriba o abajo.
padding Establece relleno a un elemento.

Propiedad Descripción

margin-top
