Selectores

Selector Descripción Identificación Utilización
Etiqueta Hace referencia a cualquier elemento que posea la misma etiqueta. <etiqueta> etiqueta{}
ID Un mismo ID solo lo debe tener un elemento por documento. <etiqueta id="name_id"> #name_id{}
Class Las clases se pueden utilizar en multiples elementos. <etiqueta class="name_class"> .name_class{}

Tipografía

Propiedad Descripción Utilización
font-family Cambia el tipo de letra del texto. selector{
font-family: name;
font-size: size (px);
color: (#hex || name);
}
font-size Cambia el tamaño de letra del texto.
color Cambia el color del texto.

Background

Propiedad Descripción Utilización
background-color Color de fondo. selector{
background-color: (#hex || name);
background-image: url(“...”);
background-position: position;
background-size: size;
background-repeat: ;
}
background-image Imagen de fondo.
background-position Posición del fondo.
background-size Tamaño del fondo.
background-repeat Repetición del fondo.

Estilos de texto

Propiedad Descripción Utilización
text-align Alineación del texto. selector{
text-align: align;
text-decoration: decoration;
text-transform: transform;
text-indent: indent (px);
letter-spacing: spacing (px);
line-height: height;}
}
text-decoration Estilos del texto.
text-transform Transformación del texto.
text-indent Identación.
letter-spacing Espacio entre letras.
line-height Espacio entre líneas.
