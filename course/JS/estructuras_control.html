Estructuras de control

Estructuras condicionales

Instrucciones que permiten dirigir la ejecución del algoritmo hacia uno u otro proceso.

if - else

Estructura de control que permite que se realice una instrucción u otra de acuerdo a la evaluación de una condición.

if (condition_1){
    statement_1
}
else if (condition_2){
    statement_2
}
//...
else{
    statement_n
}

Operador ternario

Estructura de control para condiciones simples.

condition ? expressionIfTrue : expressionIfFalse

Switch

Estructura de control que permite evaluar diferentes opciones de acuerdo a una entrada.

switch (expression){
    case value_1:
        statements
        break
    case value_2:
        statements
        break
    //...
    case value_n:
        statements
        break
    default:
        statements
        break
}

Estructuras de control iterativas

Bloque de instrucciones repetitivo hasta que se cumple una condición.

While

Estructura de control para ejecutar una secuencia de instrucciones mientras una condición sea verdadera.

while (condition){
    statement
}

For

Estructura de control para ejecutar una secuencia de instrucciones un número determinado de veces.

for (initialization; condition; afterthought){
    statement
}
