JSON

JSON es un lenguaje que permite guardar información en javascript en forma de objetos.

//Creación de un objeto persona
    const persona = {
    //Atributos,
    nombre: "Maria",
    apellido: "Suarez",
    edad: 23,
    estatura: 1.50,
    peso: 50kg,
    sexo: F,
    fechaNacimiento: "06/03/2000",
}

//Pasando de objeto en JS a formato JSON
let json_object = JSON.stringify(persona)
console.log("Objeto persona convertido a JSON:")
console.log(json_object)
//Pasando de formato JSON a objetos
let json_object = JSON.parse(persona)
console.log("Objeto persona convertido a JSON:")
console.log(json_object)
