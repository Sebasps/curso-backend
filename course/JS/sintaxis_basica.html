Sintaxis básica

En JS al terminar una linea de codigo no es necesario (;), pero no genera algun conflicto si se llega a poner.

Variables

Variables globales

Datos que pueden cambiar en cualquier lugar.

//Declaración e inicialización de variable global
var variable_name = "assigned_value"

Variables locales

Datos que solo pueden cambiar en un bloque.

//Declaración e inicialización de variable local
let variable_name = "assigned_value"

Constantes

Datos que no pueden cambiar.

//Declaración e inicialización de constante
const constant_name = "assigned_value"

Tipo de datos

Tipo Descripción Declaración
Numérico Dato que tiene asignado un valor numérico. Se declara el número sin utilizar comillas.
Lógico Dato con dos posibles valores lógicos (verdadero o falso) Se declara con las palabras reservadas true o false.
Caracter Dato que recibe cualquier tipo de texto. Se declara con comillas.
