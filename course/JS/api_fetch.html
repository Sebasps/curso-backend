API FETCH

API FETCH es una interfaz de javascript que permite realizar solicitudes HTTP.

  • GET (obtener)
  • POST (enviar)
  • PUT (actualizar)
  • DELETE (eliminar)
//Una API es un intermediario que puede ayudar en la comunicación de dos programas y así lograr compartir información
//Fetch realiza una solicitud a la pagina de JSONPlaceholder
fetch("https://jsonplaceholder.typicode.com/users")
//Una vez se recibe una respuesta por parte de JSONplaceholder, esta se guarda en response con formato JSON
.then(response => response.json())
//Por medio de data se accede a los datos
.then(data => {
    data.forEach(elemento => {
        console.log(`El nombre del usuario es: ${elemento.name}`)
    })
})

//Catch captura errores si llegan a ocurrir
.catch(e => console.error(e))

//API FETCH Para el JSON de datos
fetch("datos.json")
.then(respuesta => respuesta.json())
.then(datos => {
    console.log(datos)
    datos.forEach(persona => {
        console.log(`${persona.nombre} ${persona.apellido} tiene ${persona.edad} y su numero de telefono es ${persona.contacto.telefono}`)
    })
})
