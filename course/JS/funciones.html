Funciones

Conjunto de instrucciones que realizan una tarea especíﬁca.

Declaradas

Las funciones declaradas hacen parte del ambito global logrando ser llamadas desde cualquier parte del codigo.

//Definición de función
//La solicitud de parametros no es obligatoria
function function_name(parameters){
    statement
    //Valor que devuelve la función cuando es llamada
    return output
}

Expresadas

Las funciones expresadas hacen parte del ambito local al asignarse a una variable.

const function_name = function(parameters){
    return output
}

Flechas

Es similar a una función expresada, pero se escribe de manera aún más compacta y con una sintaxis especíﬁca.

const function_name = function(parameters) => statement
Función Ventajas Desventajas
Declarada
  • Fácil de leer y entender.
  • Puede ser llamada antes de ser declarada en el código.
  • Útil para deﬁnir funciones que se utilizarán varias veces en el código.
  • Fácilmente accesible desde cualquier parte del código.
  • No es tan versátil como las funciones expresadas o ﬂecha.
  • Puede contaminar el ámbito global de la aplicación.
  • No es posible reasignarla a otra variable.
  • No es recomendable utilizarla en casos de funciones más cortas.
Expresada
  • Fácilmente accesible desde cualquier parte del código.
  • Permite asignar una función a una variable.
  • Útil para deﬁnir funciones en el momento que se necesitan.
  • Puede ser más difícil de leer y entender cuando se tienen varias funciones expresadas en el mismo bloque de código.
  • No se puede llamar antes de ser deﬁnida.
  • No es recomendable utilizarla en casos de funciones muy largas.
Flecha
  • Sintaxis más clara y corta que la función expresada.
  • Es más fácil de usar con funciones de orden superior.
  • Útil para deﬁnir funciones cortas y simples.
  • No se puede utilizar para deﬁnir métodos en objetos.
  • Puede ser más difícil de depurar cuando hay errores de sintaxis.
  • No es recomendable utilizarla en casos de funciones muy largas.
