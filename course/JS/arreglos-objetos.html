Estructuras de datos compuestas

Arreglos

Como una colección de elementos la cual puede ser accedida por medio de un id

//Definición y creación de arreglo
const array_name = ["element_1", "element_2", ...,  "element_n"]
//Para acceder a un elemento especifico se debe proporcionar su posición
//La posición del primer elemento de un arreglo es 0
console.log(array_name[0])

element_1

//Los arreglos pueden contener diferentes tipos de datos incluyendo arreglos
const variosDatos = ["A", 1, ["cadena texto", true]]
console.log(variosDatos[2][1])

//La estructura de control for sirve para recorrer un arreglo
//Propiedad: Length permite saber cual es el tamaño de un arreglo
for (let i=0; i<variosDatos.length; i++){
    console.log(variosDatos[i])
}

//Metodos para manipulación de arreglos
//Push sirve para agregar un nuevo elemento a un arreglo
datosVarios.push(false)
console.log(variosDatos)
//Pop es el metodo que nos sirve para quitar el ultimo elemento de un arreglo
datosVarios.pop()
console.log(variosDatos)
//Splice permite eliminar elementos desde una posición y/o añadir nuevos
datosVarios.splice(1, 2, "cadena", true)
console.log(variosDatos)

//LLena las posiciones deseadas de un arreglo con un elemento determinado
const llenado = Array(5).fill(true)
console.log(llenado)

Objetos

Los objetos son estructuras de datos compuestos que tienen propiedades (atributos o caracteristicas propias del objeto) y tienen metodos (comportamiento o acciones propias del objeto)

//Definición y creación de objeto
const object_name  = {
    //Los identificadores de las propiedades pueden ser nombres, números o strings
    attribute_1: value_1,
    2: value2,
    "attribute n": value3,
    //Los metodos son comportamiento o acciones propias del objeto
    method_name_1: function(){
        return output
    },
    method_name_2: function(){
        return output
    }
}
//Acceder a los atributos de un objeto
console.log(object_name.attribute_1)
//Acceder a los metodos de un objeto
console.log(object_name.method_name_1())
