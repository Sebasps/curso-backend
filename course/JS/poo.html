POO (Programación orientada a objetos)

Objetos

Un objeto es la abstracción de algun objeto en la vida real pero en programación.

Atributos

Un atributo es una caracteristica propia de algún objeto.

Metodos

Un metodo es una acción propia que realiza algún objeto.

Clase

Una clase es una plantilla o molde que se utiliza para crear objetos similares con propiedades comunes.

//Definicion de clase
class Animal(){
    //Metodo constructor crea e inicializa un objeto creado por una clase
    constructor(especie,edad){
        //Atributos compartidos por los animales creados por la clase Animal
        //This hace referencia a la misma clase
        this.especie = especie
        this.edad = edad
    }
    hablar(){
        return "Sonido..."
    }
    moverse(){
        return "Moviendose..."
    }

//Definición de un objeto sin clases
const perro = {
    especie: "perro",
    edad: undefined,
    hablar: function(){
        return "Guau"
    }
}
perro.edad = 8
console.log(perro)

//Definición de un objeto con clases
const gato = new Animal("gato", 7)
console.log(gato)
console.log(gato.hablar())

Sonido...

//Herencia
class Perro extends Animal{
    constructor(especie, edad, nombre){
        //Metodo super llama al constructor de la clase padre
        super(especie, edad)
        this.nombre = null
    }
    //Sobreescritura de metodo
    hablar(){
        return "Guau"
    }
    saludar(){
        return "Dando la pata..."
    }

    //Setters y getters
    //Metodo set modifica algún atributo de la clase
    set setNombre(nombre){
        this.nombre = nombre
    }
    //Metodo get obtiene algún atributo de la clase
    get getNombre(){
        return this.nombre
    }
}

//Instanciar clase hijo
const labrador = new Perro("perro", 2)
//Accede a metodos de la clase padre
console.log(labrador.moverse())
//Accede a metodos de la clase hijo
console.log(labrador.hablar())
console.log(labrador.saludar())

labrador.setNombre = "Firulais"
console.log(labrador.getNombre())
